# Routes

This document lists all routes in the application and their purposes.

## Public Routes

| Route | File | Purpose |
|-------|------|---------|
| `/` | `app/page.tsx` | Home page with hero and features |
| `/docs` | `app/docs/page.tsx` | Installation and getting started |
